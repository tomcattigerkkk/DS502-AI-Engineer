# DS502-AI-Engineer

## FAQ

## Homework
作业安排请查看本次课程文件夹中homework文件夹下的README.md。

```shell
例如：第三期课程同学请进入DS502-1703文件夹，找到Homework文件夹，查看里面的README.md
```

作业提交: 每次课程后，课程老师会在Issue中开放本次作业的Issue，请讲自己的作业提交到自己的github，将链接发送到该Issue上。
